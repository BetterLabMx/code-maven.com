<a href="http://yaml.org/">YAML</a> is a data serialization format with implementations
in many programming languages. (See <a href="https://en.wikipedia.org/wiki/YAML">Wikipedia on YAML</a>.)

It is similar to <a href="/json">JSON</a>, but looks much nicer and much more readable.

It is usually used for configuration files that need more power that an INI file.

=abstract end
